Notihub
===========================

Download
---------------------------

###Generate ssh key

>	sh ./keygen.sh

Then add the public key at https://notihub.unfuddle.com/a#/people/settings

###Clone 
>	git clone git@notihub.unfuddle.com:notihub/notihub.git
>	cd notihub
>	git submodule update --init
